Conseils

La liste noire contient tous les numéros dont vous ne voulez pas être joint pour diverses raisons. Sauf si configuré autrement, aFirewall bloquera tous les numéros indiqués en liste noire, ainsi ceux étant sur cette liste, ne seront pas en mesure de vous joindre.

  • Vous pouvez configurer la méthode de blocage dans aFirewall, paramètres d'application
  • Si vous voulez, aFirewall peut vous avertir à chaque blocage. Les numéros et messages bloqués sont listé dans "appels bloqués" et "SMS bloqués"
  • Vous pouvez utiliser le numéro joker et/ou blocage de mots clés SMS. S'il vous plaît, voir les paramètres aFirewall pour plus de détails.
